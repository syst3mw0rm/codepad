CodePad

(Real time collaborative editor)

About

Recently, I've been fascinated about the engineering behind near real time, collaborative applications. Codepad is my attempt to build collaborative rich text editing application using operational transformation.

Codepad uses :

  • EditArea
  • ShareJS

The code is open source, feel free to fork and contribute.
